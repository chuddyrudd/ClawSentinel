# ClawSentinel

Scans any ClawHub or OpenClaw .md skill for dangerous code, malware, and red flags before you install it. Pure local, read-only, DataClaw trained.

## Usage

```
audit this skill: [paste skill markdown]
```

The essential safety tool every OpenClaw user needs in 2026.

## Installation

Drop `clawsentinel.md` into your `~/.openclaw/skills/` folder and restart OpenClaw.
